# NEON GRAVITY: ORBITAL

**NEON GRAVITY: ORBITAL** は、重力と幾何学が交差する、高密度な全方位シューティングゲームです。
ブラックホールによる空間の歪み、ネオンに輝くベクターグラフィックス、そして独自の「サテライト・システム」を駆使して、無限に湧き出る敵を殲滅してください。

## 🎮 ゲームの特徴 (Features)

* **Twin-Stick Action:** 移動と射撃を独立して操作する、直感的なツインスティック・スタイル。スマホとPCの両方に対応。
* **Gravity Physics:** ブラックホールや爆発の衝撃で、背景のグリッド空間がリアルタイムに歪みます。
* **Satellite System:** 敵の残骸（クリスタル）を回収して自機の周りに展開。敵弾を防ぐ「盾」として使い、ピンチの時は一斉発射して「ホーミングミサイル」に変える攻防一体のシステム。
* **Power Ups:** 赤い敵を倒して武器を強化。通常弾から2-WAY、3-WAYへと進化します。
* **Neon Visuals:** 加算合成による美しい発光表現と、大量のパーティクルエフェクト。

## 🕹️ 操作方法 (Controls)

### スマートフォン / タブレット (Mobile)
画面上に表示されるバーチャルコントローラーを使用します。

* **左スティック:** 移動 (Move)
* **右スティック:** 射撃 (Aim & Shoot)
* **LAUNCHボタン:** サテライト一斉発射 (Launch Satellites)

### パソコン (PC / Keyboard)

| Action | Key |
| :--- | :--- |
| **移動 (Move)** | 矢印キー (`↑` `↓` `←` `→`) |
| **射撃 (Shoot)** | スペースキー (`Space`) ※移動方向または最後に狙った方向へ発射 |
| **照準 (Aim)** | (移動キー入力時、その方向へ自動で照準) |
| **サテライト発射** | `Z` キー |

## 🚀 遊び方 (How to Play)

1.  **敵を倒す:** 敵を撃破すると、黄色い「クリスタル」を落とします。
2.  **回収する:** クリスタルに近づいて回収すると、自機の周りを回転する「サテライト（衛星）」になります。
3.  **盾にする:** サテライトは敵や敵弾に当たると身代わりになって消滅します。
4.  **発射する:** `LAUNCH` ボタン（または `Z` キー）を押すと、全てのサテライトが敵を追尾するミサイルとなって発射されます。ボス級の敵や、囲まれた時の打開策として有効です。
5.  **強化する:** 赤い敵を倒して `[P]` アイテムを取ると、ショットが強化されます（最大 Lv3）。

2.  `index.html` をブラウザで開くだけでプレイ可能です。
