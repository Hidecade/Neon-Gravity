# NEON GRAVITY: ORBITAL

**NEON GRAVITY: ORBITAL** は、重力と幾何学が交差する、高密度な全方位シューティングゲームです。
空間を歪ませるワームホール、ネオンに輝くベクターグラフィックス、そして攻防一体の「サテライト・システム」を駆使して、無限に湧き出る機械生命体を殲滅してください。

[👉 **Play Now (ブラウザで遊ぶ)**](https://hidecade.github.io/Neon-Gravity/)

## 🎮 ゲームの特徴 (Features)

* **Twin-Stick Action:** 移動と射撃を独立して操作する、直感的なツインスティック・スタイル。PC（キーボード＆マウス）とスマートフォンの両方に対応。
* **Gravity & Grid Physics:** 爆発やワームホールの重力干渉により、背景のグリッド空間がリアルタイムに美しく歪みます。
* **Satellite System:** 敵の残骸（クリスタル）を回収すると自機の周囲に「サテライト」が展開されます。
    * **防御:** 敵弾を防ぐ「盾」として機能します。
    * **攻撃:** 一斉発射（Launch）することで、強力な「ホーミングミサイル」として敵を追尾します。
* **Dynamic Visuals:**
    * ヒット時の火花、撃破時の派手なパーティクル拡散、壁面への着弾エフェクトなど、爽快感を重視したビジュアルフィードバック。
    * 自機のパワーアップに伴い、攻撃エフェクトが派手に進化（最大レベルで「エメラルド・フェニックス」のエフェクトが発動）。
* **Global Ranking:** Firebaseを使用したリアルタイム・オンラインランキング機能を搭載。世界中のプレイヤーとスコアを競えます。

## 🕹️ 操作方法 (Controls)

### スマートフォン / タブレット (Mobile)
画面上に表示されるバーチャルコントローラーを使用します。

* **左スティック:** 移動 (Move)
* **右スティック:** 照準 (Aim) & 自動射撃
* **LAUNCHボタン:** サテライト一斉発射 (Launch Satellites)

### パソコン (PC)

| Action | Key | Note |
| :--- | :--- | :--- |
| **移動 (Move)** | 矢印キー (`↑` `↓` `←` `→`) | |
| **射撃 (Shoot)** | スペースキー (`Space`) | 押しっぱなしで連射 |
| **サテライト発射** | `Z` または `X` | 敵を追尾するミサイルとして射出 |

## 👾 敵キャラクター (Enemies)

* **TRIANGLE:** 基本的な敵。編隊を組んで出現し、プレイヤーを取り囲みます。
* **CUBE:** アイテムキャリア。破壊すると必ずパワーアップアイテムをドロップします。
* **TADPOLE:** 素早い動きで接近してくる小型機。尾を引くエフェクトが特徴。
* **DRAGON (Armored Train):** 複数のコンテナ装甲が連結された巨大な機械竜。体をくねらせて画面を制圧します。
* **HUNTER:** 高速で接近し、弾をばら撒いて離脱するヒット＆アウェイ戦法をとる危険な敵。
* **ASTEROID:** 宇宙を漂う岩石。出現直後は無害ですが、**一定時間経過すると赤く変色して「覚醒」し、猛スピードでプレイヤーを追尾**し始めます。
* **BOSS (Geometric Fortress):** 各ステージの最後に待ち受ける巨大要塞。多角形の形状を持ち、複数の砲台から弾幕を展開します。

## 💎 パワーアップ & アイテム (Items)

緑色の敵（CUBE）や編隊を倒すと出現します。

* **[W] Weapon (Level Up):** 通常ショットが強化されます（最大Lv7）。拡散範囲が広がり、最終段階では特殊エフェクトを纏います。
* **[L] Laser:** 一定時間、敵や障害物を貫通する強力なレーザーを発射します。
* **[I] Invincible:** 一定時間、無敵状態になります。体当たりで敵を粉砕可能です。
* **[S] Shield:** シールド（HP）を回復します。
* **Crystal (緑のひし形):** サテライトを1つ補充します（最大12個）。

## 🚀 攻略のヒント (Tips)

1.  **サテライトの運用:** 常にクリスタルを回収してサテライトを維持しましょう。ボスなどの硬い敵には、サテライトを一斉発射（Launch）して大ダメージを与えるのが有効です。
2.  **アステロイドの処理:** 白い岩（Asteroid）は放置すると赤くなり、非常に危険な追尾弾となります。分裂した後も油断せず、早めに処理しましょう。
3.  **敵数制限とワームホール:** 画面内の敵が多すぎると新しい敵（ワームホール）が出現しません。処理落ちを防ぐためですが、逆に言えば敵を倒さない限り次へ進めないため、積極的な撃破が必要です。
4.  **硬い敵への対処:** ボスやドラゴンの装甲に弾が弾かれる（カキンという音と火花）場合でも、少しずつダメージは蓄積しています。諦めずに撃ち込みましょう。

## 🛠️ 技術スタック (Tech Stack)

* **HTML5 Canvas API:** 描画エンジン（加算合成、パーティクルシステム、多関節アニメーション）
* **Web Audio API:** 効果音（SE）のリアルタイムシンセサイズ生成（外部音声ファイル不使用により超軽量化を実現）
* **Firebase (Firestore):** リアルタイム・オンラインランキングシステム
* **Vanilla JavaScript:** ライブラリやフレームワークを使用しない、ピュアなJavaScriptによる実装。
