# NEON GRAVITY: DEBUGGED

**NEON GRAVITY: DEBUGGED** は、重力と幾何学が交差する、高密度な全方位シューティングゲームです。
漆黒のワームホールによる空間の歪み、ネオンに輝くベクターグラフィックス、そして独自の「サテライト・システム」を駆使して、無限に湧き出る敵を殲滅してください。

[👉 **Play Now (ブラウザで遊ぶ)**](https://hidecade.github.io/Neon-Gravity/)

## 🎮 ゲームの特徴 (Features)

* **Twin-Stick Action:** 移動と射撃を独立して操作する、直感的なツインスティック・スタイル。スマホとPCの両方に対応。
* **Gravity Physics:** ワームホールや爆発の衝撃で、背景のグリッド空間がリアルタイムに歪みます。
* **Satellite System:** 敵の残骸（クリスタル）を回収して自機の周りに展開。敵弾を防ぐ「盾」として使い、ピンチの時は一斉発射して「ホーミングミサイル」に変える攻防一体のシステム。
* **Dynamic Visuals:**
    * **Black Wormholes:** 空間を歪ませながら敵を吐き出す、鏡面仕上げの漆黒の球体。
    * **Colorful Explosions:** 敵を破壊すると、その機体色と火花が混ざり合い、光の粒子となって派手に霧散します。
    * **Bio-Luminous Enemies:** 尾を引いて泳ぐ「TADPOLE」や、立体的な編隊を組む「TRIANGLE」など、多彩な敵キャラクター。
* **Power Ups:**
    * **[L] Level Up:** ショットが拡散弾へと強化されます（最大Lv5）。
    * **[B] Beam (Laser):** 一定時間、敵を貫通する強力なレーザーを発射します。
    * **[S] Shield:** 一定時間無敵になります。
* **Instant Retry:** ゲームオーバー時、ページをリロードせずに瞬時に再スタート可能な高速リトライ設計。

## 🕹️ 操作方法 (Controls)

### スマートフォン / タブレット (Mobile)
画面上に表示されるバーチャルコントローラーを使用します。

* **左スティック:** 移動 (Move)
* **右スティック:** 照準 (Aim) & 自動射撃
* **LAUNCHボタン:** サテライト一斉発射 (Launch Satellites)

### パソコン (PC / Keyboard)

| Action | Key | Note |
| :--- | :--- | :--- |
| **移動 (Move)** | 矢印キー (`↑` `↓` `←` `→`) | |
| **射撃 (Shoot)** | スペースキー (`Space`) | 押しっぱなしで連射 |
| **サテライト発射** | `Z` または `X` | 敵を追尾するミサイルとして射出 |

## 🚀 攻略のヒント (Tips)

1.  **クリスタル回収:** 緑色の敵や編隊を倒すと「クリスタル」が出現します。回収してサテライトを常に補充しましょう。
2.  **サテライトの攻防:** サテライトは敵弾を防ぐ盾になりますが、ボス（DRAGON）などの硬い敵には `Z` キーで一斉発射して大ダメージを与えましょう。
3.  **ワームホールに注意:** 画面のあちこちに開く黒い球体（ワームホール）からは定期的に敵が出現します。囲まれないように位置取りしましょう。
4.  **アイテムの寿命:** 出現したパワーアップアイテムは3秒間はその場に留まり、その後1秒かけて小さくなり消滅します。見つけたら素早く回収してください。
5.  **密着ダメージ:** 敵に囲まれて動けなくなると、シールドが高速で削られ即座に撃墜されます。常に距離を取り、包囲網を突破しましょう。

## 🛠️ 技術スタック (Tech Stack)

* **HTML5 Canvas API:** 描画エンジン（加算合成、パーティクルシステム）
* **Web Audio API:** 効果音（SE）のリアルタイムシンセサイズ生成（mp3等の外部音声ファイル不使用）
* **Vanilla JavaScript:** ライブラリ不使用の軽量設計
